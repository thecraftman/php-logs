# Installation 

In your `env` file replace `AXIOM_API_TOKEN` with your Axiom API Token with ingesting and querying permissions, replace `AXIOM_DATASET` with your dataset name.

```
LOG_CHANNEL=axiom
AXIOM_API_TOKEN=
AXIOM_DATASET=
LOG_LEVEL=debug
```
